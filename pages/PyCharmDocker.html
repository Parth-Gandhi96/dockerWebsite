A tutorial about how to connect Docker with Pycharm to write python code - Windows

Step-1

Create pre python project in PyCharm:

Step-2

Once your python project is ready to be edited, with all dependencies in IDE fetched, create “main.py” (Or if already created then open it erase the code in it). Write the following code for printing the calender for March-2021 in console in the same file:


# Program to display calendar of the given month and year

# importing calendar module
import calendar

yy = 2021  # year
mm = 3    # month

# To take month and year input from the user
# yy = int(input("Enter year: "))
# mm = int(input("Enter month: "))

# display the calendar
print(calendar.month(yy, mm))
Step-3:

Now the python is executable in pyCharm, because all the depancies are already fetched when project was created, so you can test the code for any type of syntax code in IDE.

Step-4:

For Docker configuration in PyCharm, follow the below steps:

  • Go to File>Settings>”Build,Execution & Deployment”>Docker. And then make configuration as shown in figure below (This configuration is for Windows, please follow this step from Docker online documentation for other OS)
  • Open the Docker Engine (Docker Desktop Application), and go to settings (there is icon on the top bar for settings) - Make sure you checked every item as shown in the figure below
Step-5:

Now you have successfully attached your Docker engine with Pycharm IDE, for building-execution and Deployment. So make one new file called - “Dockerfile” (no extension). Copy the following code in the same file: 

FROM python:3

ADD main.py /

CMD [ "python", "./main.py" ]

This code will tell your Docker engine which dependencies to put inside the Docker container, and how to execute the code inside.

Step-6:

The last step before really building and running you Docker image, click on the Edit Configuration as shown in image below:

Once you click it will open a new window, expand the “template” on the left vertical bar if it's not already. Then find “Docker” expand it and then click on “Dockerfile” in the menu. Then click on the + form the right portion of the window.

After this Just give name to your image, give it a tag-id and then select the Dockerfile from dropdown. Then apply and press OK.

Step-7:

All-set! You can now press the green button to build the Dockerfile: It will open Build-log console below which will provide you the track for building the Docker image:

Once it is successfully done, go to “Log” and it will show the executed python script:
